# 新 Demo 设计总览

## 文档规则

- 这份文档用于中文镜像和玩家侧阅读。
- 英文版是维护源，这份中文文档在每次英文版有实质更新后同步刷新。
- 文档应描述当前真实可玩的状态，而不是过期计划。

## 项目概况

- 这是一个以单局突袭为核心的战术 demo。
- 玩家在局外整备，进入固定地图，在实时房间中战斗、搜索、带出物资并尝试撤离。
- 当前主入口仍然是 `project.godot -> NodeRaidDemo.tscn -> RaidMapDemo.cs`。

## 当前核心循环

1. 在局外整备。
2. 买卖、征募、升阶。
3. 选择固定地图模板和难度。
4. 进入行动。
5. 以房间视角推进，并在浮窗战略地图上规划路线。
6. 战斗、搜索、收集战利品、管理携行空间、决定继续或撤离。
7. 成功撤离或行动失败。
8. 回到局外状态。

## 当前已实现状态

### 地图与移动

- 当前使用固定地图模板，不是程序化拓扑。
- 房间通过明确的节点连线连接。
- 战略地图浮窗会随当前分辨率缩放。
- 底板地图不显示地名，只在节点上显示。
- 房间门位使用连续方向映射，不再局限于四方向或八方向吸附。
- 从一个房间的出口进入下一个房间时，会按真实反方向进入，方向关系一致。

### 房间玩法

- 房间视图是主要玩法层。
- 战斗、搜索、容器交互、房间间移动都在同一个实时房间状态里进行。
- 战略地图只是规划浮窗，不是主交互层。
- 当前房间没有敌人时，主控英雄移动速度翻倍。

### 战斗与反馈

- 房间战斗支持主控英雄、我方小兵、怪物、敌方 AI 小队。
- 近战与远程单位在视觉上已区分。
- 命中、硬直、击退已经生效。
- 近战弧光与命中特效已经重做。
- 已有程序化运行时音效：
- 轻近战，
- 重近战，
- 轻远程，
- 重远程，
- 轻受击，
- 重受击，
- 死亡。

### AI 呈现

- AI 小队使用橙黄暖色。
- 普通怪物和威胁单位使用灰白色。
- 死亡 X 标记会继承单位颜色。

### 容器与搜索

- 容器已从右侧边栏迁移到房间场景中。
- 容器会在非战斗状态下于近距离自动打开。
- 搜索目前只允许在非战斗状态下进行。
- 搜索仍然基于容器格子、揭示、拿取这套流程。
- 搜索面板是半透明浮窗，不再是全屏压暗弹框。
- 玩家打开容器后仍然可以移动。
- 离开当前容器范围会中断搜索并关闭面板。
- 容器已有 `拾取全部`，只会拿走当前已可拿的物品。

### 死亡掉落

- 普通单位死亡不再固定生成尸体容器。
- 普通非英雄单位会掉落资源球。
- 资源球当前只提供金钱。
- 主控英雄会自动吸附并收取附近资源球。
- 本局收取到的金钱会单独显示在侧栏。
- 普通敌方单位有低概率额外掉落特殊容器。
- 我方普通小兵不会掉普通尸体容器。
- 精英和类 Boss 死亡仍会生成独立精英容器。
- 背包丢弃会在房间里生成临时弃置容器。

## 携行空间与队伍背包

### 当前规则

- 局内背包已经不是无限列表。
- 物品从容器进入背包时会保留原始尺寸。
- 容量占用和容量上限会显示在 UI 中。

### 当前容量来源

- 当前没有单独的基础容量块。
- 英雄容量块：`2 x 2`
- 每个普通小兵容量块：`1 x 1`
- 当前容量来源取决于局内仍存活的我方单位。

### 当前拼接规则

- 容量不是一个完整大矩形，而是由多个容量块自动拼接。
- 当前 UI 每行上限是 `12` 格。
- 不可用格不会显示。
- 容量来源之间会用虚线分隔，帮助理解来源。
- 这些视觉分块不构成硬边界，物品仍可跨块放置。

### 当前增长规则

- 容量按 `4 x 4` 区域分段生长。
- 当前区域会优先朝更接近方形的方向补齐。
- 同分时按从上到下、从左到右选择。
- 当前 `4 x 4` 有效补满后，才会往右开启下一块区域。

### 当前背包交互

- 已支持拖起物品。
- 已支持旋转。
- 已支持悬停名称。
- 较大的物品会直接绘制名称。
- 已有自动整理按钮。
- 自动整理会同时处理背包区和待整理区物品，并允许旋转。

### 超载与待整理

- 新物品放不下时，系统会先尝试自动整理。
- 因队友死亡导致容量下降时，也会先自动整理。
- 仍放不下的物品会进入右侧 `待整理` 区。
- 待整理区有物品时视为超载。
- 超载时不能过门。
- 超载时不能撤离。
- 超载不影响当前房间内移动。
- 可通过丢地面来解超载。

## 局外、仓库与经济

- 局外买卖已可用。
- 招募系统已可用。
- 当前招募得到的都是 `新兵`。
- 初始测试资金目前是 `3000`。
- 仓库已经是固定网格。
- 当前仓库尺寸：`12 x 10`
- 商店买入会真正尝试放入仓库。
- 仓库支持选中、出售、重排。
- 局外仓库与局外背包之间已支持拖放。
- 撤离后不会自动把局内物资塞进仓库。
- 当前流程是：结果页 -> 结算转移页 -> 玩家手动决定哪些进仓库。

## 小兵成长现状

### 当前第一波成长链

- 小兵现在会持久保存 `兵种 + 经验`。
- 当前已落地兵种：
- `新兵`
- `盾兵`
- `精锐盾兵`
- `钢盔盾卫`
- `壁垒盾卫`
- `枪兵`
- `精锐枪兵`
- `钢盔枪卫`
- `御锋枪卫`
- `刀兵`
- `精锐刀兵`
- `钢盔刀卫`
- `断阵刀卫`
- `弓兵`
- `精锐弓兵`
- `钢盔猎手`
- `穿杨射手`
- 局外整备页已有士兵 roster 面板。
- 新兵可以在局外面板中升阶。
- 升阶会消耗金钱并要求经验。
- 幸存士兵会在清房和成功撤离时获得经验。

### 盾兵线

- 盾兵线当前是四段：
- `盾兵`
- `精锐盾兵`
- `钢盔盾卫`
- `壁垒盾卫`
- 基础盾兵只有被动 `飞矢防护`。
- `精锐盾兵` 获得主动 `盾冲`。
- `钢盔盾卫` 获得头盔并强化防守被动。
- `壁垒盾卫` 获得更华丽的大盾和更强的盾冲冲击波表现。

### 枪兵线

- 枪兵线当前也是四段：
- `枪兵`
- `精锐枪兵`
- `钢盔枪卫`
- `御锋枪卫`
- 基础枪兵只有被动 `拒马列阵`。
- `精锐枪兵` 获得主动 `挺枪突刺`。
- `钢盔枪卫` 获得头盔，并强化 `拒马列阵`。
- `御锋枪卫` 获得更华丽的枪刃，并把主动强化为更宽、更长、可穿透多目标的 `贯阵突刺`。
- 枪兵实际攻击距离已经比普通近战更长，并与视觉枪长做了同步。
- 枪兵的突刺姿态已改成更平滑的过渡，不再突然跳过去。

### 其他兵种

- 刀兵线当前也是四段：
- `刀兵`
- `精锐刀兵`
- `钢盔刀卫`
- `断阵刀卫`
- 基础刀兵只有被动 `追猎`。
- `精锐刀兵` 获得主动 `突进斩`。
- `钢盔刀卫` 获得头盔，并强化 `追猎`。
- `断阵刀卫` 获得更华丽的刀，并把主动强化为带范围劈斩的 `断阵突袭`。
- 弓兵线当前也是四段：
- `弓兵`
- `精锐弓兵`
- `钢盔猎手`
- `穿杨射手`
- 基础弓兵只有被动 `鹰眼`。
- `精锐弓兵` 获得主动 `分裂箭`。
- `钢盔猎手` 获得头盔，并强化 `鹰眼`。
- `穿杨射手` 获得更华丽的弓，并把主动强化为更密集的多目标齐射。

- 主控英雄
- 次要英雄

- 小兵成长已经起步，
- 主控英雄与次要英雄成长还没正式落地。

- 英雄成长
- 次要英雄成长
