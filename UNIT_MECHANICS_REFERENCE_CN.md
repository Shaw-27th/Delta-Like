# 单位机制参考

## 文档范围

- 这份文档记录当前游戏里真实生效的单位数值与战斗规则。
- 它应与当前可运行 demo 的实现保持一致。
- 英文版是维护源，这份中文文档是给玩家和策划查看的镜像。
- 每次英文版有实质更新后，这份中文文档也应同步刷新。

## 全局战斗规则

- `HP`：当前与最大生命值。
- `Damage`：攻击时在区间内随机取值。
- `Armor`：在特殊倍率之后结算的固定减伤。
- `AttackRange`：近战或远程的交战距离。
- `Speed`：基础移动速度，不含技能覆盖。
- `AttackCycleScale`：攻击节奏倍率，越低越快。
- `Stamina`：跑动和主动技能会消耗体力。
- `ProjectileDamageScale`：远程伤害在计算护甲前先乘上的倍率。
- `BlockAnyDamageChance`：有概率在伤害结算前直接完全格挡。

## 小兵通用技能规则

- 大部分非远程、非特化兵种默认主动技能是 `跑动`。
- 弓兵不跑动。
- 盾系兵种不跑动。
- 枪系兵种不跑动。
- 盾系基础被动：
- `飞矢防护`：对远程伤害减半。
- 枪系基础被动：
- `拒马列阵`：在攻击距离外缘命中时，造成更强的击退与硬直。

## 盾冲规则

- 当前使用者：
- `精锐盾兵`
- `钢盔盾卫`
- `壁垒盾卫`
- 冷却：`5s`
- 体力消耗：`24`
- 基础触发条件：
- 目标距离大于 `AttackRange + 18`
- 目标距离小于等于 `150`
- 施放者当前可行动且体力足够
- 基础冲刺：
- 持续时间 `0.28`
- 移动速度 `520`
- 路径效果：
- 会把沿途单位撞开，
- 会对敌我双方产生物理推开，
- 同阵营单位只会被推，不会受伤
- 终点效果：
- 比路径接触更强的击退与控制

### 壁垒盾卫的强化盾冲

- 路径和终点影响半径更大
- 伤害更高
- 硬直更强
- 击退力度与持续时间更高
- 带有沿路线展开的冲击波特效
- 冲击波只做视觉表现，不额外增加碰撞

## 挺枪突刺规则

- 当前使用者：
- `精锐枪兵`
- `钢盔枪卫`
- `御锋枪卫`
- 冷却：`5s`
- 体力消耗：`18`
- 基础触发条件：
- 目标距离大于自身攻击距离的 `72%`
- 目标距离小于等于 `AttackRange + 26`
- 施放者当前可行动且体力足够
- 基础机制：
- 不是位移冲锋，
- 使用独立的起手和收招，
- 沿前方一条窄直线进行突刺
- 精锐枪兵与钢盔枪卫：
- 命中直线上的最近一个有效目标
- 御锋枪卫：
- 直线更宽，
- 直线更长，
- 可穿透多个目标，
- 带更强的路线波纹特效

## 刀兵突进斩规则

- 当前使用者：
- `精锐刀兵`
- `钢盔刀卫`
- `断阵刀卫`
- 冷却：`4.5s`
- 体力消耗：`18`
- 基础触发条件：
- 目标距离大于 `AttackRange + 8`
- 目标距离小于等于 `96`
- 施放者当前可行动且体力足够
- 基础机制：
- 短距离突进后接重斩
- 作为贴身破阵手段，而不是普通跑动
- 被动联动：
- `追猎` 会对残血或已受控目标增伤并加重打击感
- 强化版 `追猎` 触发时还会返还一部分攻击冷却

### 断阵刀卫的强化突进斩

- 突进距离更长
- 伤害更高
- 击退与硬直更强
- 会顺带劈到主目标附近的敌人
- 带短促的冲击波特效

## 分裂箭规则

- 当前使用者：
- `精锐弓兵`
- `钢盔猎手`
- `穿杨射手`
- 冷却：`5s`
- 体力消耗：`16`
- 基础触发条件：
- 目标距离大于自身攻击距离的 `46%`
- 目标距离小于等于 `AttackRange + 18`
- 施放者当前可行动且体力足够
- 基础机制：
- 保持远程站位，不做位移
- 一次攻击会射向主目标，并额外分裂至最多 `2` 个附近目标
- 被动联动：
- `鹰眼` 会让远程攻击有概率暴击
- 强化版 `鹰眼` 会进一步提高暴击率与暴击伤害

### 穿杨射手的强化分裂箭

- 最多 `3` 个额外目标
- 搜索范围更大
- 次级箭伤害更高
- 整体齐射打击感更强

## 士兵经验规则

- 幸存士兵在清空一个房间时获得 `+1 XP`。
- 幸存士兵在成功撤离时再获得 `+1 XP`。
- 死亡士兵会从本局 roster 中移除，不会继续获得经验。
